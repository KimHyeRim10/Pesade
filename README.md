## 주제 선정
기존의 Pesade라는 향수 사이트를 단일 브랜드의 향수만 판매하는 것이 아니라, 다양한 브랜드의 향수를 판매하는 플랫폼으로 확장한 클론 코딩 프로젝트입니다.

## 작업 기한
- 2024.06 - 2024.07 (5인)

## 맡은 역할
- 로그인, 회원가입, 아이디찾기, 비밀번호찾기, 회원정보 수정
